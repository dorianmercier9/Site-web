Exercice 1

Crée une liste appelée nombres contenant les nombres suivants : 16, 92, 120, 60, 21.

Utilise une boucle for pour parcourir la liste nombres.

À chaque itération de la boucle, affiche le nombre en cours.

Assure-toi que les nombres sont affichés dans l'ordre de la liste.

nombres = for num in nombres: nombres = [16, 92, 120, 60, 21] for num in nombres: print(num) 16 92 120 60 21 sct = ''' ex() %>% check_object("nombres") %>% check_equal() '''
Assure-toi d'initialiser correctement la liste nombres avec les nombres donnés. Ensuite, utilise une boucle for pour parcourir la liste et afficher chaque nombre à chaque itération.

Exercice 2

Initialise une variable number à la valeur 0.

Utilise une boucle while pour afficher les nombres tant que number est inférieur à 50.

À chaque itération de la boucle, affiche la valeur actuelle de number.

Si number atteint la valeur 30, utilise l'instruction break pour sortir de la boucle.

Assure-toi que les nombres sont affichés dans l'ordre croissant, avec un incrément de 10 à chaque itération.

number = 0 number = 0 while number < 50: print(number) if number == 30: break number = number + 10 sct = ''' ex() %>% check_object("number") %>% check_equal() '''
Assure-toi que l'expression dans la condition de la boucle while est correcte pour s'assurer que la boucle se termine lorsque number atteint la valeur 30. Vérifie également que tu utilises l'instruction break lorsque number atteint cette valeur pour sortir de la boucle.
